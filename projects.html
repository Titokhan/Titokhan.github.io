Open Source Projects

speakerboxxx

A slack application to ChatOps enable your GitHub for organizations with multiple teams and channels.

slash-heroku

A slack application to ChatOps enable your heroku interactions

Hubot

Hubot is a chat robot for most popular chat networks.

Camo

A simple SSL proxy for images. Useful for making insecure images embedded in websites appear to come from a secure resource.

warden-github

Authenticate with GitHub in [rails](https://github.com/fphilipe/warden-github-rails) or [sinatra](https://github.com/atmos/sinatra_auth_github).

Auto-Deploy

A GitHub service for auto-deployment workflows. Use conventions in git collaboration and have the computers work for you.

Heroku Beta

A GitHub service for deploying code to Heroku.

OpsWorks Deployment

A GitHub service for deploying code to Amazon OpsWorks.
